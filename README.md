## 🥗 IT Healthy - Frontend Contribution (React)

## 📝 Project Overview
IT Healthy is a team-based e-commerce web application focused on healthy food ordering.  
The platform allows users to explore products, manage their cart, and complete orders through a smooth and user-friendly interface.

👉 Original Repository: https://github.com/TLHSinh/IT_Healthy

---

## 🎯 My Role
Frontend Developer (React)

In this project, I was mainly responsible for building the client-side application and ensuring seamless interaction between the user interface and backend services.

---

## 🚀 What I Did
I focused on developing a modern, responsive, and dynamic user interface using React and Tailwind CSS.

- Built reusable React components for key features such as product listing, cart, and checkout
- Integrated RESTful APIs to fetch and manage data (products, cart, orders)
- Implemented dynamic UI updates using API calls (Axios) without full page reload
- Designed responsive layouts to ensure compatibility across devices
- Improved user experience through smooth interactions and clean UI structure

---

## 🛠 Tech Stack
- ReactJS (Component-based architecture)  
- Tailwind CSS (UI styling & responsive design)  
- JavaScript ES6+  
- Axios (API integration)  
- RESTful API  

---

## 💡 Key Features I Worked On

- Shopping cart interaction  
- Checkout flow UI
